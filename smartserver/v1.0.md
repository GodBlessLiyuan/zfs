## 砖柱智能小助手接口文档 ##

***
- 所有的请求消息都是JSON格式
- 所有的相应消息

***

***
### 错误码 ###
	- 1000 返回正确结果
	- 1001 json格式不合法
	- 1002 app版本不支持
	- 1003 参数缺失
	- 1004 用户token过期
	- 1005 用户非VIP
	- 1006 VIP用户
	- 1007 年费VIP用户
	- 1008 最新版本
	- 1009 需要更新
	- 2000 服务器内部错误
***

### 接口说明 ###

#### 1. 设备请求接口 ####
	- url: http://server_addr/smart/v1.0/device
	- method:POST
	- 请求消息示例：
	```
		{
			"androidid":"",
			"utdid":"",
			"osv":"",
			"channel":"",
			"factory":"",
			"model":"",
			"softv":1,
			"uuid":"",
			"imei":["imei1","imei2","meid"]
		}	
	```
	参数说明：
	```
		androidid：androidid唯一标识
		utdid：阿里唯一标识
		osv：系统版本号
		channel：应用渠道
		factory：手机厂商
		model：手机型号
		softv：应用版本号
		uuid: Android端随机生成的字符串
		imei：设备imei或meid
	```
	响应消息及错误码：
	```
		 {
			"status":1000,
			"data":{
				"id":"deviceid",
				"verify":"deviceid md5"
			}
		}
	```
	参数说明：
	```
		code: 返回状态
		id: 设备唯一标识
		verify: 设备唯一标识的md5值
	```

#### 2. 获取短信验证码接口 ####
	- url: http://server_addr/smart/v1.0/sms
	- method:POST
	- 请求消息示例：
	```
		{
			"id":int,
			"ph":"188****2717",
			"verify":"deviceid md5"
		}	
	```
	参数说明：
	```
		id: 设备唯一标识
		ph: 手机号码
		verify: 设备注册接口返回的md5值
	```
	响应消息及错误码：
	```
		{
			"code":1000
		}
	```
#### 3. 用户注册接口 ####
	- url: http://server_addr/smart/v1.0/register
	- method: POST
	- 请求消息示例：
	```
		{
			"sms":"",
			"id":int,
			"verify":"",
		}
	```
	参数说明：
	```
		sms：短信验证码
		id：设备唯一标识
		verify：设备唯一标识md5，设备接口返回数据
	```
	相应消息及错误码：
	```
		{
			"code":1000,
			data:{
				"ud":1,
				"um":"********"，
				"token":"****"
			}
			
		}
	```
	参数说明：
	```
		code：返回码
		ud：用户唯一标识int类型
		um：用户唯一标识的变种md5
		token: 用户签名验证
	```

#### 4. 用户反馈信息接口 ####
- url: http://server_addr/smart/v1.0/feekback
- method:POST
- 请求消息示例：
	```markdown
		{
			"ud":int,
			"id":int,
			"um":"",
			"verify":"",
			"context":""，
			"picdata":"",
			"factory":"",
			"model":"",
			"softv":1,
			"osv":"",
		}	
	```
	
	参数说明：
	
	```markdown
		ud: 用户唯一标识id
		um: 用户唯一标识id的md5
		id: 设备唯一标识
		verify: 设备注册接口返回的md5值
		context: 反馈内容
		picdata: 图片数据的base64
		factory：手机厂商
		model：手机型号
		softv：应用版本号
		osv：系统版本号
	```
	
	响应消息及错误码：
	
	```
		{
			"code":1000
		}
	```
	

#### 5. 上传异常log ####
	- url: http://server_addr/smart/v1.0/exception
	- method:POST
	- 请求消息示例：
	```
		{
			"id":int,
			"md":"",
			"osv":"",
			"softv":int,
			"model":"",
			"exception":"",
			"pkg":""
		}	
	```
		参数说明：
	```
		id: 设备唯一标识
		m: 设备注册接口返回的md5值
		os_version: android系统版本
		soft_version: 软件版本号
		model: 手机型号
		exception: 异常日志
		pkg: 包名
	```
	响应消息及错误码：
	```
		{
			"code":1000
		}
	```

#### 6. 验证用vip状态 ####


	- url: http://server_addr/smart/v1.0/check
	- method:POST
	- 请求消息示例：

```
	header  token:token
	{
		"id":int,
		"verify":"",
		"ud":int,
		"md":"",
		"uuid":""
	}	
```
参数说明：
```
	id: 设备唯一标识
	verify: 设备注册接口返回的md5值
	ud: 用户唯一标识id
	um: 用户唯一标识id的md5
	uuid: 软件初始安装生成的uuid
```
响应消息及错误码：
```
	{
		"code":1007,
		"data":{
			"vip":"2017-09-08 16:36:20",
			"super":"2017-09-08 16:36:20"
		}
	}
```
参数说明：
```
	code: 返回状态码 1005 非VIP、1006 VIP用户、1007 年费VIP用户
	vip: 用户vip过期时间
	super: 年费会员vip过期时间
```

#### 7. 应用更新 ####

	- url: http://server_addr/smart/v1.0/check
	- method:POST
	- 请求消息示例：

```
	{
		"id":int,
		"verify":"",
		"softv":int
	}	
```
参数说明：
```
	id: 设备唯一标识
	verify: 设备注册接口返回的md5值
	softv: 应用版本
```
响应消息及错误码：
```
	{
		"code":1008
	}
	
	{
		"code":1009,
		"data":{
			"url":"",
			"md5":""
		}
	}
```
参数说明：
```
	code: 返回状态码 1007 最新版本、1008 需要更新 
	url: 更新应用下载链接地址
	md5: 下载应用的md5值
```

#### 9. 获取开屏广告配置信息  ####

- url: http://server_addr/smart/v1.0/adconfig
	- method:POST
	- 请求消息示例：

```
	{
		"id":int,
		"verify":"",
		"softv":int
		"channel":"huawei"
	}	
```

参数说明：

```
	id: 设备唯一标识
	verify: 设备注册接口返回的md5值
	softv: 应用版本
	channel: 应用渠道
```

响应消息及错误码：

```
	{
		"code":1000,
		"data":{
			"strategy": 3, 
			"default":"",
			"ads":[
                "adid": 1，
                ”source“: 1,
                "sort": 1,
                "number":"",
                "starttime":"",
                "endtime":"",
                "link":"",
                "icon":""
			]
		}
	}
```

参数说明：

```
	code: 返回状态码 1000
	strategy: 广告策略
	default: 默认广告背景图的下载地址
	adid: 广告id
	source: 广告来源 自有  第三方
	sort: 优先级
	number: 后台配置广告id
	starttime: 广告开始时间
	endtime: 广告结束时间
	link: 点击链接地址
	md5: 插件的md5值
```
